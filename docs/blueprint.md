# **App Name**: Bilingual News Hub

## Core Features:

- Bilingual Article Publishing: Create and publish articles in both English and Gujarati, with language-specific URLs and a language toggle feature.
- Article Display: Server-rendered article pages with structured data, social sharing buttons, related stories, and a comment section with moderation.
- Admin Dashboard: A modern admin dashboard for creating/editing articles, scheduling content, managing reporters, issuing press-cards, and viewing donations. Only the owner can access articles.
- Donation System: One-time and recurring donation options via Razorpay (India) and Stripe/PayPal (global), with automated receipts.
- Reporter Galaxy: A directory of reporters with unique QR codes, added by the admin and visible to all users.
- RSS Widget Integration: Integrate RSS feeds to display news from other sources.
- Login/Signup System: Authentication system for owners and users.
- User Profiles: System for users to create and manage their profiles, including saving contact information.
- Navigation System: Intuitive navigation system for easy access to different sections of the website.
- Language Toggle Button: Button to switch between English and Gujarati languages.
- Ad Container: Ad containers for advertisements, designed to be non-intrusive and visually appealing.
- Theme Selector: Theme selector allowing users to choose between light, dark, or system default themes.

## Style Guidelines:

- Primary color: Dark Blue (#0B6EFD), for a clean, trustworthy feel inspired by livemint.com, nytimes and thehindu.com.
- Background color: Light gray (#FAFAFA) to ensure a whitespace-forward, content-first design.
- Accent color: A vibrant shade of Blue Violet (#4A148C) to create contrast with main brand color.
- Additional theme colors: Yellow, White, and Blue, with options for users to select a theme (dark, light, or system default).
- Headline font: 'Playfair' (serif) for headlines to offer an elegant and high-end feel. Proper typography considerations for readability and visual hierarchy will be implemented.
- Body font: 'PT Sans' (sans-serif) as a counterpart for long texts.
- Space allocation for logo placement in the header, ensuring brand visibility.
- Simple, clear icons for navigation and actions, ensuring ease of use.
- Clean editorial grid layout, resembling livemint.com, nytimes and thehindu.com, with a hero section and well-structured content sections.
- Subtle animations for transitions and loading states to enhance user experience without distraction.
